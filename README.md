# The Brenicks Group — Website

**AI Implementation. Salesforce. Web Design.**

Boutique AI implementation, Salesforce, and web design consulting for ambitious businesses.

---

## ✨ Features

- Light mode (Daytime Côte d'Azur) + Dark mode (Cannes After Midnight)
- Theme preference saved to localStorage
- Glassmorphism cards with Lamborghini door-glass precision borders
- Smooth scroll reveal animations
- Responsive — mobile, tablet, desktop
- Marquee service strip
- Cursor glow on desktop
- No frameworks — pure HTML/CSS/JS

---

## 🎨 Design System

**Typography:** Playfair Display (headings) + Satoshi (body)

**Light Palette:** Ivory `#FAF8F4` · Champagne `#F3F0EA` · Aquamarine `#8EC5D0` · Gold `#B8956A`

**Dark Palette:** Obsidian `#080C14` · Deep Navy `#0D1320` · Warm Gold `#C8A96E`
